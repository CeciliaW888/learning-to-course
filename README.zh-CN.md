# Learning to Course

一个 Claude Code 技能，能将**任何学习主题**转化为结构化的 PWA 课程，包含每日学习指南、互动测验和 GitHub 仓库。

告诉它你想学什么，就能得到一套完整的课程——离线可用的网站、Excalidraw 图表、闪卡和经过验证的学习资源。

## 适合谁？

任何想学新东西、希望有系统学习路径而非零散教程的人。

**实用导向：**
- 每天有清晰的学习计划
- 互动测验考察实际应用，而非死记硬背
- 可安装的离线应用
- 跨天跨周跟踪学习进度

适用于任何主题：AI 智能体、烹饪、乐理、云架构、语言学习——你想学什么都行。

## 课程长什么样

输出是一个 **GitHub 仓库 + 可安装的 PWA 网站**：

- **每日学习指南** — 800-1500 字的 Markdown 文件，每篇包含 8 个必备部分
- **可安装 PWA** — 离线可用、可添加到主屏幕、追踪进度
- **互动测验** — 场景化问题、闪卡、拖拽匹配
- **代码 ↔ 中英对照** — 真实代码配上通俗解释
- **动画可视化** — 数据流动画、组件对话动画
- **术语提示** — 悬停任何技术术语即可查看通俗定义
- **Excalidraw 图表** — 源文件 + 导出的 SVG
- **验证过的资源** — 每个 YouTube 链接经 oEmbed 验证，每个文档链接已检查

### 多语言支持

课程可以生成**英文**、**简体中文**或**双语**版本。中文课程使用思源宋体、中文标点，并提供本地化资源（B站、小红书）。

## 如何使用

### 快速安装

```bash
npx skills add CeciliaW888/learning-to-course
```

### 手动安装

1. 将 `learning-to-course` 文件夹复制到 `~/.claude/skills/`
2. 打开 Claude Code
3. 输入：*"把机器学习做成一个两周的课程"*

### 触发短语

- "帮我创建一个关于 X 的课程"
- "把 X 变成课程"
- "给我做一个 X 的学习路径"
- "帮我做一个 X 的课程大纲"
- "我想学 X——帮我制定学习计划"

### 它会问你什么

生成前，技能会问你 6 个问题：

1. **学什么？** — 你想学什么主题？
2. **什么水平？** — 零基础、有一定了解、还是有经验？
3. **每天多久？** — 30 分钟 / 1 小时 / 2 小时？
4. **学多长时间？** — 1 周 / 2 周 / 3 周 / 1 个月？
5. **什么语言？** — 英文、简体中文还是双语？
6. **学完想达到什么目标？**

课程长度是**动态的**——快速入门可能 5 天，深度学习可能 30 天。

## 设计理念

- **展示，而非讲述** — 每个页面至少 50% 是视觉内容，每段文字最多 2-3 句。
- **考应用，不考记忆** — 不问"API 的全称是什么？"而是问"用户反馈数据不更新，你会先检查哪里？"
- **只用验证过的资源** — 每个 YouTube 视频都经 oEmbed 验证，绝不使用虚构的链接。
- **新鲜的比喻** — 每个概念都有独特的比喻，绝不重复使用。

## 互动元素

课程包含 17 种互动元素：

| 元素 | 说明 |
|------|------|
| 选择题测验 | 场景化问题，每个选项有独立反馈 |
| 代码 ↔ 中英对照 | 代码与通俗解释并排展示 |
| 术语提示 | 悬停/点击查看技术术语的通俗定义 |
| 聊天动画 | iMessage 风格的组件对话 |
| 数据流动画 | 逐步展示数据在系统中的流动 |
| 闪卡 | 3D 翻转卡片，带学习进度追踪 |
| 拖拽匹配 | 触屏友好的概念匹配练习 |
| 找 Bug | 点击代码中有问题的那一行 |
| 提示框 | 高亮"顿悟时刻"和关键洞见 |
| 步骤卡 | 可视化的流程分步展示 |
| 文件树 | 带说明的目录结构展示 |
| 层级切换 | 渐进式 HTML → CSS → JS 展示 |
| 架构图 | 可交互的系统架构总览 |
| 模式卡片 | 工程模式/概念的网格展示 |
| 反思问题 | 保存到本地存储供回顾 |
| 进度追踪 | 连续学习天数、测验分数、完成度 |
| 完成庆祝 | 测验通过时的撒花动画 |
